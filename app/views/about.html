What is Security Base?

Security-Base goal is to regroup in a single place all forums, online tools, irc channels, blogs etc directly or indirectely related to information security. This project started as a sort of super bookmarkt for myself and I thought it would be nice to have this online and try to get some contributions.

Contribute

If you want to contribute to this project feel free to submit a pull request on either the frontend repository or the backend repository on Github
