What is Security Base?

Security-Base aim to to regroup in a single place all forums, irc channels, blogs, tools, etc directly or indirectely related to security / infosec. This project started as a sort of super bookmarkt for myself and I put it online in the hope that people will contribute to it and add more bases :)

Contribute

If you want to contribute to this project you can add links or make a pull request on either the frontend repository or the backend repository on Github. If you have other ideas or you want to create a bot that will scrape twitter for infosec blogs and feed them to Security Base or something like that then feel free to shoot me a tweet or an email. There is a tag system in place but the data are not tagged. So even if you're not a programmer and don't have links you can still help.
